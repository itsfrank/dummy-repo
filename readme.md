# CLI Project

A simple Rust CLI application with three commands: foo, bar, and baz.

## Usage

```bash
# Run the foo command
cargo run -- foo

# Run the bar command
cargo run -- bar

# Run the baz command
cargo run -- baz
```

## Commands

- **foo**: Prints "foo command was run"
- **bar**: Prints "bar command was run"
- **baz**: Prints "baz command was run"
